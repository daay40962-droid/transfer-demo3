## 0. Главное в двух словах

Делаем **демо-лендинг** пассажирских перевозок по маршруту
**Красноярск — Северо-Енисейский**. Цель этой версии — **показать заказчику**,
что мы попали в тематику и стиль. Не финальный продукт, а рабочий шоурум:
выглядит и ведёт себя как настоящий сайт. Реквизиты уже реальные
(ИП Ежкова Екатерина Михайловна, ИНН 243401091643), а MAX-бот и тексты
юридических политик — заглушки, которые подключим/заменим после одобрения.

Конечную точку маршрута везде пишем строго: **Северо-Енисейский**.

> **Порядок сборки:** сайт делаем **по частям, не весь сразу**. Этапы и статусы —
> в `ROADMAP.md`. После каждого этапа — показ и согласование.

## 1. Технические решения

- **Тип:** одностраничный лендинг (one-page) с якорной навигацией (плавный скролл
  к блокам, без отдельных страниц).
- **Принцип:** **mobile-first**. Большинство зайдёт с телефона — мобильная версия
  главнее десктопной. Крупные кнопки, удобно нажимать пальцем, быстрая загрузка.
- **Стек:** чистый HTML + CSS + JavaScript (без тяжёлых фреймворков), чтобы сайт
  летал на мобильном интернете. Данные (точки, цены, контакты) вынесены в
  отдельный data-объект `siteData` — меняются без правки вёрстки.
- **Хостинг для демо:** выкладываем на временную бесплатную ссылку (например,
  Netlify / Vercel / GitHub Pages), чтобы заказчик открыл с любого телефона.
- **Видео-фон:** НЕ используем. На главном экране — статичная картинка
  (`Gemini_Generated_Image_dmvzyxdmvzyxdmvz.png`).

## 6. Каналы связи и формы

На сайте **4 канала связи** + 2 формы.

### 6.1. Кнопки прямой связи
| Кнопка | Поведение | Технически |
|---|---|---|
| **Звонок** | открывает набор номера | ссылка `tel:+79233465997` ✅ |
| **SMS** | открывает СМС с готовым текстом-шаблоном | ссылка `sms:` с `body` ✅ (на телефоне) |
| **MAX** | открывает чат с диспетчером | ссылка `max.ru/<профиль>` ✅ |

> Важно про MAX: предзаполненный текст в личный чат MAX вставить **нельзя**
> (мессенджер не поддерживает). Кнопка просто открывает чат с диспетчером.
> Шаблон-приветствие возможен только у СМС.

**Шаблон SMS:**
> Здравствуйте! Пишу с сайта по поводу бронирования поездки
> Красноярск — Северо-Енисейский.

### 6.2. Форма обратного звонка
Поля: **Имя**, **Телефон**.
Текст рядом:
> Оставьте номер — мы перезвоним и уточним детали поездки.
Обязательный чекбокс согласия на обработку персональных данных (без него не отправляется).

### 6.3. Форма онлайн-заявки
Поля:
- Имя
- Телефон
- Дата поездки
- Желаемое время
- **Направление** (выпадающий список точек)
- **Туда / обратно** (переключатель)
- Комментарий
- Чекбокс согласия (обязателен)

**Автоподстановка:** при клике на точку карты или на карточку цены — поле
**Направление** заполняется автоматически, страница скроллит к форме.

### 6.4. Куда уходят заявки
- **В демо:** логику доставки «под капотом» НЕ делаем (решение по вопросу 30).
  Формы просто проверяют заполнение + согласие и показывают экран
  **«Спасибо, заявка принята, диспетчер свяжется с вами»**. Никуда реально не
  отправляются. Этого достаточно для показа клиенту.
- **В боевой версии:** заявки уходят в **MAX через бота** (Max Bot API). Для этого
  нужно верифицировать реальное ИП (Ежкова Е.М.) в dev.max.ru — без этого бот в MAX
  не публикуется. Почта (email) НЕ используется.

## 13. Расписание и услуга «от двери до двери»

- **Приём заявок:** круглосуточно.
- **Отправления:** утренний сбор ~5:00–8:00, вечерний сбор ~17:00–20:00.
- Объяснить: это не минута в минуту, а период сбора; точное время согласуется
  индивидуально по адресам.
- Заметно вынести: **«Забираем от двери и привозим до двери»**. Для Красноярска —
  из любой точки города (квартира, дом, гостиница, вокзал, автовогзал).
