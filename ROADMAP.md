# ROADMAP — сборка сайта по этапам

## Главный принцип
Сайт собираем **по частям, не весь сразу**. После каждого этапа — показ Коляну
и согласование. Следующий этап начинаем только после «ок» по предыдущему.
Так раньше видно направление и дешевле развернуть, если что-то не так.

Источник правды по содержанию — `tz-FINAL.md`. Этот файл — только про порядок работ.
